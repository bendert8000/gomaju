** 功能特色 **

休息規則
    — 任意數量、自訂休息間隔、休息長短；全螢幕+多螢幕，讓你不得不休息! 但是必要情況還是可以跳出休息繼續作業。
    — 可暫停計時或立刻休息，暫停太久也會提示是否繼續休息計時。
    — 可以在不同的休息畫面顯示自定義文字，比如說你想做的伸展運動或護眼20-20-20法則。
    — 可以自訂休息開始、結束時的音效提醒。

鬧鐘 — 自訂名稱+說明，靈活重複方式：單次、每日、每週、每2週、每月、每年。

計時器 — 常用的倒數計時預設（最長 99:59:59），開始後倒數到 0，也可切換成正數計時；內建多組常用預設（1／3／5／10／15／30 分、1 小時）。計時中可顯示即時的倒數小窗與進度條，時間到會跳出提示窗並持續累計超時、播放提示音。

碼錶 — 精準到百分之一秒的碼錶（最長 99:59:59.99），可記錄分段（Lap）。支援開始／暫停／重設，空白鍵快速開始／暫停；開始／暫停時可選擇播放音效提醒。

自訂提示音 — 用於休息提醒以及鬧鐘，可以用音符自己編一段旋律或匯入音檔（wav／mp3／ogg／flac）。

休息小語 — 休息畫面可以顯示一句勵志小語，已內建十多則各國中英文小語，支援自定義。

休息前提醒 — 快開始休息了，會先跳一個不搶焦點的小倒數（要提前幾秒可以自己設；填 0 就是不要）。也可以在倒數時臨時延後休息1分鐘。

閒置不計時 — 電腦閒置2分鐘就會自動暫停計時。

跨平台、省資源 — Windows、MacOS、Linux 均可用。可設定開機自動啟動。啟動快，實測僅需幾 MB 記憶體。

安全 — 全部資料設定都存在用戶本機，隱密性佳。

高度自定義 — 大部分行為都有參數可修改。如果沒有也可以你修改程式碼後自行編譯使用。
