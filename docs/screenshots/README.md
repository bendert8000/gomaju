# Screenshots

These images are embedded in the top-level [`README.md`](../../README.md). Drop the four
PNGs below into this folder (exact filenames):

| File | What to capture |
| --- | --- |
| `break-overlay.png` | A break screen. **tray → Break now** (a Soft rule shows the green soft overlay), then snip; dismiss with the Skip affordance or by holding `Esc`. |
| `break-rules.png` | The **Break-rules dashboard** — open from **tray → Breaks…**. |
| `alarms.png` | The **Alarms** window — open from **tray → Alarms…** (ideally with the **Bi-weekly** repeat row expanded so the start-date + weekday picker shows). |
| `settings.png` | The **Settings** window — open from **tray → Settings…**. |

Capture tips:
- **Use the English UI:** **tray → Language → English**, then (re)open each window so it
  renders in English.
- **Window only**, not the whole desktop: `Win`+`Shift`+`S` (region snip) or `Alt`+`PrtScn`
  (active window).
- Keep them reasonably tight to the window so they look crisp at the README's widths
  (~250 px for the three windows, ~720 px for the overlay).
